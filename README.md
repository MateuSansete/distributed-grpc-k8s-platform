# distributed-grpc-k8s-platform
Projeto de microsserviços distribuídos que compara o desempenho de gRPC com REST-API para um agregador de itinerários de viagem. Apresenta uma arquitetura poliglota (Node.js/Python e Go) implantada no Kubernetes. Focado na orquestração de pipelines de dados, serialização de protobuf e análise comparativa de padrões de comunicação de microsserviços.
